写在前面

​ 从该博客开始，将回顾前几个月的MySQL学习过程，由于平时科研紧张，因此只能晚上抽出几小时来学习，目前只是过了一遍SQL的基本命令，了解了一些聚合函数、join以及子查询等。那些更深、更专业的SQL知识，还待进一步学习与掌握。

安装部分

​ 对于该部分，鉴于各个学习者的安装环境（windowsX、Linux）、版本的不同，本博客暂不对具体安装过程做出描述，可能会在博客撰写过程中，对该部分进行一个补充。

概念肃清

平时，熟知或者听说的数据库管理系统就不少，例如：商业:oracle, SQLserver，DB2；开源:MySQL， postgreSQL， SQLite。虽然它们的SQL语句会有出入，但是学会了一种SQL语句，其他的自然很快会上手，语言不是问题，这句话不仅适用于此。

MySQL的10个基本语句

当安装结束、环境搭建成功后，正式开始学习MySQL。

1、启动服务：net start mysql （此处的mysql是标识符，取决于你安装过程中所取的名字）。登入：mysql -uroot -p，然后在输入密码。或者mysql -uroot -p,但这样易造成密码泄露，推荐前者。

2、查看有哪些库，show databases;（分号是必需的，初学易忘记）

一开始在没有创建新库时，会有几个固有数据库，请不要在这些库中做改动。

3、use name; (这里的name就是show 后的库名)

4、show tables; （顾名思义，查看该库下所有表）

5、CREATE DATABASE [IF NOT EXISTS] <数据库名>
[ [DEFAULT] CHARACTER SET <字符集名>] [ [DEFAULT] COLLATE <校对规则名>];

[]表示可选

  1. IF NOT EXISTS：在创建数据库之前进行判断，只有该数据库目前尚不存在时才能执行操作。此选项可以用来避免数据库已经存在而重复创建的错误;

  2. CHARACTER SET：指定数据库的默认字符集。可用charset简写

    有哪些字符集后续会介绍；

  3. COLLATE：指定字符集的默认校对规则。

最简单的示例：CREATE DATABASE ;

6、删除数据库：drop database ku_name;

7、数据库不能改名，表可以改名

8、删除表，drop table tables_name;

9、表改名：rename table xxx to new_xxx;

10、表中插入数据，insert into 表名 values;

11、清空表数据：truncate 表名; 与delete的区别，前者是删表再重建一张同样结构的表，而delete是删除表中的数据。清空速度上，前者更快。
